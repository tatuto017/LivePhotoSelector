# Ollamaによる振り分け仕様

## 処理概要
Ollamaで写真の振り分けを行う

## 要件
- 複数人写っているのは対象外
- 楽器等は持っておらず、特徴は髪色と髪型
  - かい(金髪ショート)
  - おと(黒髪ショート)
  - みりん(黒髪ツインテール)
- 照明の変化が激しいライブなので精度重視

## 処理フロー
- `{PROJECT_ROOT}/input`から写真を読み込む
- `{PROJECT_ROOT}/output`に振り分けを行う
- 複数人の写真は`{PROJECT_ROOT}/output/複数人`に振り分けを行う
- 自分が不明な写真は`{PROJECT_ROOT}/output/不明`に振り分けを行う
